Learnbook

This is a SCORM 1.2 compatible learning module template that is loaded with features. It creates navigation, does branching, ad hoc page-sets created by the user, allows you to embed questionmark or captivate quizzes Simple javascript interactions Scores the user based on what percent they achieved of total possible maximum points from all quizzes.
