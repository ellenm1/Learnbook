Another LMS
If this is for an external LMS, like ACEMAPP:
  1. The launch url and paths to all assets in the manifest must be relative urls ("index.htm"), not root-relative ("/content/ct.."). Using this form should help you do this correctly.
  2. Remove the default index.htm file. Rename the file "index-local.htm to "index.htm". "index-local" has relative urls for all assets.
  3. Our questionmark quizzes will not work, since they require access. Replace with other types of quizzes.
